VARIÁVEIS

Variaveis são espaços na memória do computador que armazenam valores. São chamadas de var, const ou let.
Elas são como vagas e guardam informações que podem ser usadas no programa.
Exemplo:
- var nome = 'João'
- let idade = 20
- const cidade = 'São Paulo'
Importante lembrar que eu não posso REDECLARAR uma variável que já existe.
Eu posso mudar seu valor, mas não posso criar outra com o mesmo nome.
EM CONSTANTES NAO POSSO MUDAR nada, NEM MESMO SEU VALOR.
Evitar usar VAR e sim LET no lugar

DATA TYPES

Essas variáveis podem conter vários tipos de informações como:

  • Strings = texto entre aspas
  • Numbers e dentre eles podemos ter (infinitos, NAN, inteiros, parciais)
  • Booleans = true ou false
  • Null = não possuem valores
  • Objects{} = contem alguns valores acessados por nomes
  • Arrays[] = alguns valores dentro acessados por índices (0 a infinito)

PRIMITIVOS E REFERÊNCIA

Primitivos = não podem ser alterados em si (não tem a ver com a variável e sim com o valor que ela guarda) -> String, Number, Boolean, Null e etc. (Quando eu usar o valor de uma variável deste tipo em outra os valores serão copiados e independentes entre si.)
Referenciados = podem ser alterados em si (não tem a ver com a variável e sim com o valor que ela guarda) -> Arrays, Objects e Functions (Quando eu usar o valor de uma variável deste tipo em outra os valores serão alterados caso eu altere uma delas. Isso só não ocorrerá se eu usar o spread operator (...) para copiar os valores.)

Exemplo: const a = 'joão' const b = a Se eu alterar o valor de a posteriormente o b não se alterará pois é um valor primitivo.
Se eu fizer o mesmo com os mutáveis, e alterar o valor de a, o b também se alterará pois é um valor referenciado.
Para que ele nao se altere neste último caso eu escreveria b como const b = {...a}. Aí ele se tornaria um valor primitivo e independente de a.

n1.toString()

OPERADORES

Permitem realizar operações com as variáveis e possuem alguns tipos como:

  • Aritméticos (), **, *, /, %, + e - (nessa ordem de precedência)

    • % é o resto da divisão inteira (o que sobra antes de começar a virgula na divisão) e ** é potencia (numero elevado)
  • atribuição - São substituições que eu faço na minha operação (como escalonamento em matemática)
  • Relacionais - >, <, >=, <=, == e != (resultam em boolean = verdadeiro ou falso) (posso fazer essas comparações com strings tb) e não possuem precedência.
  • Identidade - identificam se uma valor é exatamente igual ao outro. Pra isso eu uso ===
  • Lógicos - ! é negação, && (um e outro) conjunção (eu preciso que os dois sejam verdadeiros) e ||(um ou outro) é disjunção (um ou outro precisa ser verdadeiro) seguindo esta ordem de precedência

    • exemplificando tudi em uma função:
    salário >= 2000 && sexo != 'mulher' --> o salário é maior ou igual a dois mil e o sexo não é uma mulher. Ou seja, homens que ganham no mínimo 2000
  • Ternário - ? e : dentro de uma mesma operação - Eu faço uma pergunta e defino as opções que podem aparecer se for verdadeiro ou falso, nesta ordem. O primeiro valor é se a resposta for sim e o segundo se for não
    Exemplo:
    var média = 7 -> media >= 8? 'aprovado':'reprovado'

CONDIÇÕES

As condições são usadas para tomar decisões no código.
Exemplo:
Se eu quero que o meu programa faça algo se uma condição for verdadeira, eu uso o if.
Nesse caso trata-se de uma condição simples, pois se o resultado for falso nada acontece.
Para criar uma condição composta, eu uso o else.
Então if é verdadeiro e else é falso.
A estrutura basica de uma condição é a seguinte:
if (condição) {
//código a ser executado se a condição for verdadeira
else {
//código a ser executado se a condição for falsa

CONDIÇÕES ANINHADAS

Basicamente, são condiçoes dentro de condições.
Entao eu faço o if e dou uma condição, se for verdadeira irá executar, se for falso irá abrir outra condição com mais duas condições dentro e assim por diante
Posso usar tambem o else if no lugar de abrir um novo if e em seguida somente o else.
ex: else if (condição) {
//código a ser executado se a condição for verdadeira
}
else (não pdoe ter condição, porque é tudo o que sobrar) { //código a ser executado se as anteriores nao forem verdadeiras }

REPETIÇÕES

Repetições, ou laços, são usadas para executar um bloco de código várias vezes.
para isso temos o while, o do while e o for.

O while é uma estrutura de repetição que executa um bloco de código enquanto uma condição é verdadeira.
Exemplo:
while (condição) {
//código a ser executado enquanto a condição for verdadeira

O do while é uma estrutura de repetição que executa um bloco de código enquanto uma condição é verdadeira.
Basicamente é o mesmo do while, mas ele executa o código pelo menos uma vez antes de verificar a condição.
Exemplo:
do {
//código a ser executado enquanto a condição for verdadeira
while (condição);

O for é uma estrutura de repetição que executa um bloco de código um número determinado de vezes.
Exemplo:
for (inicialização; condição; incremento) {
//código a ser executado enquanto a condição for verdadeira

}
Se eu defino que var c é 1 e que c é menor ou igual a 10, e que c++ eu estou dizendo que o valor de c vai ser 1 e depois vai ser 2 e assim por diante até chegar a 10.

Ex for (var c = 1; c <= 10; c++) {
//código a ser executado enquanto a condição for verdadeira
console.log(c)
}
Se eu quiser criar um array e usar o for para percorrer ele, eu posso fazer assim:
var numeros = [1, 2, 3, 4, 5]
for (var i = 0; i < numeros.length; i++) {
console.log(numeros[i])
}
Então o for vai percorrer o array numeros e imprimir cada número na tela.
OBS: sempre coloco menor sem o igual porque o length do array é o número de elementos que ele tem, e se eu colocar igual ele vai tentar acessar um elemento que não existe, já que os arrays começam do 0.

É uma variável que pode armazenar vários valores em um único espaço de memória.
Isso ocorre quando eu defino uma variavel X e dentro dela tenho varias "vagas" começando por 0 e terminando no número que eu definir.

Nome é o Array e dentro dele temos as vagas 0, 1 e 2. que são os nomes que escolhi.
ACÕES:

  • Vejo quantos índices tenho e digo variavel [proximo numero] = x.
  • variável.push para que ele mesmo adicione o número na proxima vaga
  • .unshift para adicionar no início ao inves do final
  • .length para descobrir quantos índices
  • .sort para ordenálos
  • .reverse para inverter a ordem
  • .indexOf para descobrir um elemento no array
  • .pop eu removo o último elemento
  • .shift eu removo o primeiro elemento
  • delete variável [índice] remove o que eu quiser e deixa o espaço vazio
  • .slice (de um indice até o que eu quero) vai considerar o intervalo
Exemplos:
var num = [5, 8, 10]
num.push(7)
ai ele adicionará o numero 7 em seguida dentro do array.
Exemplo:
var num = [5, 8, 10]
num.indexOf(8)
ele vai me retornar o numero 1, pois o numero 8 está na vaga 1.

Se eu quiser que ele retorne outra coisa que nao seja o -1 padrão, eu posso usar o if (se o numero for igual a -1) {

DESESTRUTURAÇÃO

A desestruturação é uma forma de extrair valores de um objeto ou array e atribuí-los a variáveis individuais.
Exemplo:
const pessoa = {nome: 'João', idade: 30}
const {nome, idade} = pessoa
console.log(nome) // João
console.log(idade) // 30
Eu posso usar o mesmo conceito para arrays.
Exemplo:
const numeros = [1, 2, 3]
const [a, b, c] = numeros
console.log(a) // 1
console.log(b) // 2
console.log(c) // 3

exemplo:
